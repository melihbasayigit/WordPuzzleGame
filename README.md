<h2>🧐 Features</h2>

Here're some of the project's best features:

*   58000 Turkish words
*   Undo your selection
*   Swibe left or right to check the selection
*   60 Fps support
*   Stop/Start game button
*   Save the score
  
<h2>💻 Built with</h2>

Technologies used in the project:

*   Kotlin
*   Android Studio
*   SQLite
*   Paralize
*   Gesture Detector
*   Written in accordance with OOP rules
*   CustomView for game view
*   Multithread
